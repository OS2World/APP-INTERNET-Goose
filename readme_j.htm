Goose
    Gooseはいわゆるアップデートされたホームページをチェックするユーティリティです。

    GooseはVX-REXXで作成されていますので、実行にはVX-REXXのRuntime Library `VROBJ.DLL'が必要となります。 `VROBJ.DLL'は下記をクリックするかまたはBBSやOS/2の雑誌等から入手することができます。

    • Download VROBJ.DLL 2.1d (vrobj21d.zip)

Install
  • インストール

    1. gse???r.zipを適当なディレクトリにコピーし、解凍します。

    2. install.cmdを実行するとGooseのオブジェクトがデスクトップに作成されます。

Usage
  • URLの登録

    1. 新規登録
      新規にURLを登録する場合、`New'ボタンをクリックしてください。

    2. 登録データの修正
      登録されたデータを修正する場合、`Edit URL'ボタンをクリックしてください。

    3. メモの追加・修正
      メモの追加・修正をする場合、`Edit Memo'ボタンをクリックしてください。

    4. 登録サイトのパラメータ

      • URL1
        チェックしたいURLを記述してください。

      • URL2
        チェックしたいホームページがフレームを使用している場合、チェックしたいURLをURL1に、OpenしたいURLをURL2に記述してください。

          URL1:Set check URL
          URL2:Set open URL

        Note: この時、`Open Secondary'を選択している必要があります。

      • Open Primary/Open Secondary
        URL2を使用時、どちらのURLを開くか選択可能です。

      • Memo
        MemoにURLの詳細を記述することができます。 Memo使用時、Gooseはタイトル表示の横にMemoを表記することができます。
        Note: プリファレンスで`Show Memo'を選択する必要があります。 URL2を使用時は`Set Memo as title when using secondary URL'を選択する必要があります。

  • 機能

    1. Context Menu

      • Edit

        1. New Site
          新しいサイトを登録します。

        2. Edit Site
          選択しているサイトのデータの変更します。

        3. Edit Memo
          選択しているサイトのメモの追加・修正を行います。

        4. Delete Site
          選択しているサイトを削除します。

      • URL

        1. Copy URL
          選択しているURLをクリップボードにコピーします。

        2. Open URL
          選択しているURLを開きます。

        3. Check URL
          選択しているURLをチェックします。

    2. `Check Previously Not Updated Site Only'
      前回、Updateされていないサイトだけをチェックする場合、Runボタンを押す前に`Check Previously Not Updated Site Only' check boxをチェックしてください。

    3. `Skip'
      特定のサイトのチェックをスキップすることができます。 一度Enabledに設定したサイトは次回Disabledされるまで保持されます。
      Note: コンテキスト・メニューからチェックを行った場合、SkipパラメータはDisabledされます。

  • Preference

    1. Proxy

      • Proxy/Portを設定する
        Proxyの使用をお薦めします。

      • Ping Proxy
        常にProxy Serverの状態をチェックする場合は、`Ping Proxy Every...'を選択します。 チェックのインターバルも設定できます。

      • Proxy Siteの追加・修正・削除

    2. URL List

      • Type
        リストの表記方法(タイトル表記またはURL表記)を設定します。

      • Style
        リストの表示スタイル(テキストまたはグラフィカル)を設定します。 グラフィカルを選択時、背景に画像を表示することができます(固定)。

      • Memoの表示
        タイトル横にメモを表記する場合は`Show Memo'を選択します。

      • Set Memo as title when using secondary URL
        この機能はURL2使用時の`Show Memo'機能と同等です。

    3. 一般設定 General

      • Remember last selected site list
        Goose終了時に選択していた表示モードを記憶させたい場合、`Remember last selected site list'を選択します。

      • Change to updated view when check is done
        URLのチェック終了後、自動的に表示モードを`Updated'に切り替える場合は、`Change to updated view when check is done'を選択します。

      • Font
        フォントの設定をします。

    4. HTML File (Goose.htm)
      Gooseはチェック終了後、その結果をGoose.htmというファイルを作成します。

      • Create Goose .htm Shadow on Desktop.
        Goose.htmのシャドウをデスクトップに作成する場合、`Create Goose.htm Shadow on Desktop'を選択します。

      • Background Image
        Goose.htmで使用するバックグラウンド・イメージを設定します。

      • Color
        Goose.htmで使用する色を設定します。

      • Date and Time
        Goose.htmでの日時の表記方法を設定します。

        例
      • Normal (24Hour)
        		• :28 Oct 1999 22:55
      • Ordered (12Hour)
        		• :99/10/28 10:55pm
      • USA (24Hour)
        		• :10/28/99 22:55
      • European (12Hour)
        		• :28/10/99 10:55pm

    5. Homepage Browser
      ブラウザを設定します(Netscape Navigator 4.6でのみテストを行っています)。

    6. Auto Start
      Gooseの起動時にURLのチェックを自動的に開始させる事ができます。 チェックを開始するまでのインターバルも1〜30秒の範囲で設定できます。

  • Note

    1. 幾つかのサーバーからのデータが取り込めません。

    2. Proxy Serverの利用をお薦めします。
