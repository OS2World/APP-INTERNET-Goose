Goose
    Goose is one of `Checking Updated Homepage' utility.

    This OS/2 application requires VROBJ.DLL, the runtime module from Watcom VX-REXX.
    You can get a copy from...

    • Download VROBJ.DLL 2.1d (vrobj21d.zip)

Install
  • Install

    1. Unzip the archived file into any unique local directory.

    2. Run INSTALL.CMD by either OS/2 full screen or OS/2 session. The Goose program object will automatically be created on your desktop.

Usage
  • Register Site

    1. Register new site
      Push `New' button to add new site.

    2. Edit site data
      Push `Edit URL' button to edit URL at first.

    3. Edit memo
      Push `Edit Memo' button to edit memo at first.

    4. Site parameters

      • URL1
        Set target URL.

      • URL2
        When the target site uses FRAMEs, you can set check URL and open URL separately.

          URL1:Set check URL
          URL2:Set open URL

        Note: At this time, you should select `Open Secondary' radio button.

      • Open Primary/Open Secondary
        When you use URL2 parameter, select primary or secondary URL which you want to open.

      • Memo
        You can set the simple description for the site. When you use memo, Goose can shows memo in place of title at Title Area.
        Note: Check `Show Memo' check box at Preference. When you use URL2, then check `Set Memo as title when using secondary URL' check box.

  • Functions

    1. Context Menu

      • Edit

        1. New Site
          Register new site.

        2. Edit Site
          Edit URL of selected site.

        3. Edit Memo
          Edit memo of selected site.

        4. Delete Site
          Delete selected site.

      • URL

        1. Copy URL
          Copy selected URL to clipboard.

        2. Open URL
          Open selected URL.

        3. Check URL
          Check selected URL.

    2. `Check Previously Not Updated Site Only'
      When you want to check previously not updated site only, check `Check Previously Not Updated Site Only' check box before push `Run' button.

    3. `Skip'
      You can set the site that you want to skip to check. This parameter is kept till you set disabled.
      Note: After you check the site from context menu `Check Site', `Skip' parameter is disabled.

  • Preference

    1. Proxy

      • Use Proxy/Port
        I recommend to set proxy server.

      • Ping Proxy
        Check `Ping Proxy Every...' check box when you want to check proxy server is active. Also you can set interval time (minutes).

      • Add/Edit/Delete Proxy Site

    2. URL List

      • Type
        Select title type from Title or URL.

      • Style
        Select URL list window style from Graphic or Text only. When you select Graphic, you can indicate background image (Goose default image only).

      • Show Memo
        Check `Show Memo' check box when you want to show memo at title field.

      • Set Memo as title when using secondary URL
        This parameter is same as `Show Memo' when you uses URL2. Check `Set Memo as title when using secondary URL' check box when you want to use show memo at title field.

    3. General

      • Remember last selected site list
        Check `Remember last selected site list' check box, when you want to remember last selected view mode when Goose startup time.

      • Change to updated view when check is done
        Check `Change to updated view when check is done' check box, when you want to change view mode to `Updated' when check is done.

      • Font
        Of course, you can change font as you like.

    4. HTML File (Goose.htm)
      Goose create summary file, Goose.htm after checking homepages.

      • Create Goose.htm Shadow on Desktop.
        Check `Create Goose.htm Shadow on Desktop' check box when you want to create Goose.htm shadow on the desktop.

      • Background Image
        You can set background image of Goose.htm file.

      • Color
        You can set color set of Goose.htm file.

      • Date and Time
        You can select Date and Time style which are written in Goose.htm file.

        For example...
      • Normal (24Hour)
        		• :28 Oct 1999 22:55
      • Ordered (12Hour)
        		• :99/10/28 10:55pm
      • USA (24Hour)
        		• :10/28/99 22:55
      • European (12Hour)
        		• :28/10/99 10:55pm

    5. Homepage Browser
      You can set homepage browser you like (I tested only by Netscape Navigator 4.6).

    6. Auto Start
      You can start checking URLs at Goose start up time. Also you can set interval (1-30 sec).

  • Note

    1. Goose cannot get data from some sites(servers).

    2. For more stability, I recommend to use Proxy Server.
